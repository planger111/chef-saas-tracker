Security & Privacy
INFRA SalesPlay Tracker · Managed by Infra Sales

Authentication

This application requires authentication via Microsoft Entra ID (Azure Active Directory) using your Progress Software account. Only employees with a valid Progress Microsoft 365 identity may access this application.

Single Sign-On is enforced. You cannot create a local password or access the app without your Progress SSO credentials.

Data Storage

All data is stored in Microsoft SharePoint within the Progress Software Microsoft 365 tenant. No data is stored outside the Progress tenant.

  • Engagement logs and account assignments are stored in SharePoint document libraries
  • Rep identity data is stored in SharePoint and never committed to any public code repository
  • Activity logs (login/action events) are stored in SharePoint, admin-only access

Access Control

  • Reps — can view their own accounts, log engagements, and see the leaderboard
  • Managers — can view all engagement data across their team
  • Admins (Owners) — can configure plays, upload accounts, view activity logs, and manage all data

Admin access requires explicit authorization by Infra Sales leadership. Role assignments are hardcoded in the application and changes require a code deployment.

Activity Logging

All user actions — including logins, form submissions, screen views, and demo mode usage — are logged with timestamp, user email, and action type. These logs are accessible only to application admins and are stored in SharePoint.

Hosting & Infrastructure

  • Hosted on Azure Static Web Apps in the Progress Azure subscription
  • Source code hosted on GitHub (public repo — code only, no data or credentials)
  • No server-side logic — all reads/writes go through Microsoft Graph API using your authenticated session token
  • No third-party analytics, tracking, or external services

Confidentiality

This application and its data are Progress Software proprietary and confidential. Do not share access, screenshots, or data with anyone outside of Progress Software. Do not embed or transmit engagement data to external systems.

Contact

For access requests, security questions, or to report an issue, contact Infra Sales leadership.
